Applied Technology Magnet

ATM, Applied Technology Magnet, is a 3 year computer program that is only offered at Bancroft middle school for middle school students in the Long Beach Unified School District. This program will teach you a little bit about every thing, from computer graphics, to CAD, to Multimedia. Here are the subjects you will be learning in all 3 years:

6th Grade Learning

  • Typing
  • Microsoft Word
  • Microsoft Powerpoint
  • Microsoft Excel
  • Coding Concept Basics: Javascript

7th Grade Learning

  • Adobe Photoshop
  • Adobe Illustrator
  • SketchUp
  • Floorplanner

8th Grade Learning

  • Filming planning & camera Shots
  • Garageband
  • Imovie
  • Coding: HTML & CSS

My opinion

ATM is a very interesting program to be in. It introduced many things to students about computers that you did not even think you could do. I would say that my 7th and 8th grade years were definitely the most fun I had in this program. This was only because I always wanted to learn how to code and use photoshop & illustrator. If you really like using computers and want to accel your skills in technology, then I would definitely recommend this program. ATM really covers almost a little bit of everything so that you get a head start on some programs. This is so that you can figure out if you want to improve in the software; or have knowledge on a software, so that you will not struggle with it in the future.
